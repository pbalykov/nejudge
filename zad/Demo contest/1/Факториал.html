Факториал

Напишите программу, которая вычисляет факториал натурального числа N ( 1 ≤ N ≤ 100 ), сохранив все значащие цифры.
Факториалом называется произведение всех натуральных чисел от 1 до N : N ! = 1·2·3... ( N - 1)· N .

Входные данные

Входная строка содержит натуральное число N ( 1 ≤ N ≤ 100 ).

Выходные данные

Программа должна вывести факториал числа N .
